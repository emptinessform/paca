# Paca 문서 — 한국어 번역

이 폴더는 저장소 루트의 [`docs/`](../../docs/) 문서 중 **리더 중심 핵심 문서**를 한국어로 옮긴 것입니다. `docs/` 트리 구조를 그대로 미러링했습니다.

- 각 문서 상단에는 번역 안내 배너가 있습니다.
- **코드·명령·경로·URL·링크 대상은 원문 그대로** 유지했습니다 (번역 대상은 자연어 산문뿐).
- API 스펙(`api/http-design.md`)·DB 스키마·SDK 레퍼런스 등 **밀도 높은 기술 레퍼런스는 영어 원문 유지**가 실용적이라 제외했습니다 — 원문은 [`docs/`](../../docs/)에서 확인하세요.

> ℹ️ 일부 문서(`guides/design-system.md`, `ai-agent/overview.md`)는 문서 내 목차(TOC) 앵커 링크가 깨지지 않도록 **섹션 제목을 영어로 유지**하고 본문 산문만 번역했습니다.
